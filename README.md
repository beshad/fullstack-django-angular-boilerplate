# Fullstack Django 2 and Angular Boilerplate

> Server
- Django
- PostgreSQL database
- Port: 7000

> Client
- Angular
- Port: 3000 

##### Install dependencies
```pipenv shell```
```npm install```
```pipenv install```


##### Run: 
```npm run server``` & ```npm run client``` 
(or start both:  ``` npm start```)
